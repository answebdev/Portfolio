Front-end Projects
The Simpsons Clicky Game

The Simpsons Clicky Game is a React application that tests your memory with a Simpsons-themed memory game. The user tries to click on all twelve cards without clicking the same card twice. The cards are shuffled each time the user clicks on a card. How good is your memory?

Bar Finder

The first project I worked on with a team. This web application utilizes the Google Maps API and two other APIs to allow a user to search for a bar based on location and bar type. Users may also search for drink recipes so that they may make the drinks themselves. By creating an account and logging in, users can select bars and drinks and add them to their Favorites.

 Deployed  GitHub Repo
Train Scheduler

An application that provides up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station. The time of the train's next arrival is calculated in relation to the current time, which is powered using Moment.js. Firebase is used to host arrival and departure data.

GifTastic

This dynamic web page populates with gifs using the GIPHY API. As the user searches for a topic, the page populates with buttons by topic. When each topic button is clicked, ten gifs are displayed. The user can change the state of each gif from "still" to "animate", and vice versa, by clicking on the gif.

This game combines HTML and CSS techniques with JavaScript and jQuery. Try to match the random number by clicking on the four different crystals, each with its own random value set at the start of play. Flash messages display when the user wins or loses.

Back-end Projects
LIRI Bot

LIRI Bot is a command line Node app that takes in parameters and returns data. Through the use of APIs, users can search Spotify for songs, Bands In Town for concerts, and OMDB for movies. Log your search results in a separate .txt file to save what you have searched for. This application employs the use of NPM packages, including Moment, Request, and the Node-Spotify-API package.

 Video Demo  GitHub Repo
Bamazon

Bamazon is a command line Node app that uses MySQL. A Customer View takes in orders from customers and depletes stock from the store's inventory, while a Manager View allows managers to view a list of the products for sale, view low inventory, add to the inventory, and add a new product. This application employs the use of NPM packages, including MySQL, Inquirer, and the Cli-Table package.

Full-stack Projects
Friend Finder

Friend Finder is a compatibility-based application meant to simulate a simple dating app. This full-stack app takes in results from a user's response to questions and compares his or her answers with those from other users. The app will then display the name and picture of a user with the best overall match. Friend Finder uses Node.js and Express on the back-end and Bootstrap on the front-end for the design.

 Deployed  GitHub Repo
Eat-Da-Burger!

"Eat-Da-Burger!" is a restaurant app that allows users to input the names of burgers they'd like to eat. Burgers are kept in the waiting area until they are devoured, after which they are placed in the "Burgers that have been devoured" section. This full-stack app uses MySQL, Node, Express, Handlebars, and a homemade ORM, and follows the MVC design pattern.

 Deployed  GitHub Repo
Music Blog

Music Blog is an Express app that allows the user to enter and save album information on any band or artist. The user may enter the band/artist name, the name of the album, and the track listing. The user can also choose to edit or delete his or her entries. This application uses RESTful routes and MongoDB to store user data. Semantic UI is used for the design.
