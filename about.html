About Me

Welcome to my website

I am a web developer based in San Diego, California. I enjoy creating clean and dynamic web applications. As a full-stack developer, I utilize my diverse set of skills to create projects involving both front-end and back-end technologies. As a user-focused developer, I stand for quality and focus on details in order to produce and develop great web apps and user-friendly experiences. I have a passion for the design end of applications as well as working with the back end. Feel free to explore my website and connect with me!



Front-end

My front-end skills include HTML, CSS, JavaScript and jQuery. In addition, I am able to use other technologies like Bootstrap, Materialize, and Semantic UI to create responsive sites. Some of the front-end applications that I've developed are GifTastic, The Simpsons Clicky Game (React), and Bar Finder, a collaborative effort created on a team of four. Check out the portfolio section for more information on these apps.


Back-end

My coding interests extend beyond the front-end. The languages and technical skills in my back-end toolbelt include Node.js, Express.js, NPM, Handlebars, REST, as well as MongoDB and MySQL databases. Applications highlighting these skills are LIRI Bot and Bamazon, which can be found in the portfolio section. Head on over and take a look at the demo videos I created for these applications.


Full-stack

Developing full-stack applications means I can create and develop a fully operational website, server and all, from scratch. I am highly adept with the MERN stack and RESTful routes, and can work on projects using the MVC design pattern. Full-stack applications that I've developed include Friend Finder and Eat-Da-Burger, both found in the portfolio section of my website. Stop on by and have yourself a burger!
